# Geolocation

You can find your location on the Yandex map.

## Installation

Clone this repo to your local machine using https://github.com/AleksandrPuhov/geolocation.git

Use the npm package manager.

```npm
$ npm install
```

## Usage

Use the npm package manager to build.

```npm
$ npm run build
```

```npm
$ npm run build:dev
```

```npm
$ npm run build:prod
```
